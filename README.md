# COVID-19-Global-Public-Health-Analysis
Analyzed 49,000+ global COVID-19 records (over 16.4 million cases) for trend analysis.Calculated 5 core public health metrics using Python/Pandas. Implemented an ARIMA(1, 2, 1) forecasting model to predict 7-day confirmed case trends. Provided strategy to monitor outbreak kinetics and guide short-term public health response.
